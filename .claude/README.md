# Claude Code Configuration

This directory contains Claude Code configuration for the DCG Parsing Skill project.

## Skills

### dcg-parsing

Located in [skills/dcg-parsing.md](skills/dcg-parsing.md), this skill guides Claude in writing efficient, idiomatic SWI-Prolog DCGs (Definite Clause Grammars).

**To use this skill in a conversation:**
```
/skill dcg-parsing
```

or through the Skill tool in the SDK.

## Project Structure

The skill includes comprehensive documentation:

- **[SKILL.md](../SKILL.md)** - Core patterns and philosophy
- **[PATTERNS.md](../PATTERNS.md)** - Comprehensive DCG patterns library (30+ patterns)
- **[ERROR_HANDLING.md](../ERROR_HANDLING.md)** - Error handling and debugging strategies
- **[EXAMPLES.md](../EXAMPLES.md)** - Complete integrated examples
- **[TESTING.md](../TESTING.md)** - Testing strategies and debugging
- **[REFERENCE.md](../REFERENCE.md)** - Performance considerations, common mistakes, and references
